1) Модифицируйте сервер Express, чтобы он выполнял следующие действия:
Возвращал файл index.html по маршруту /.
Возвращал файл student.html по новому маршруту /student.
Возвращал JSON-ответ по маршруту /api/info, который содержит следующий объект: { "name": "Ваше Имя", "task": "Динамические маршруты Express", "status": "В процессе" }
Убедитесь, что сервер обрабатывает ошибки корректно.
Если файл не найден, возвращайте статус 404 Not Found и сообщение об ошибке.
2) Добавьте параметр запроса к маршруту /api/info, чтобы можно было динамически обновлять поле "status".
Например: /api/info?status=Completed должен возвращать: { "name": "Ваше Имя", "task": "Динамические маршруты Express", "status": "Completed" }
